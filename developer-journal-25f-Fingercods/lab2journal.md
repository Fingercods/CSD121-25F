class method: a function you call using the class name. example Math.random().

instance method: a function you call on an object you already made. example frame.setSize(200,200).

class variable: a constant or value that belongs to the class itself. example Color.BLACK.

instance variable: a value or property that belongs to one object. example styles.length.

constructor:  uses new to make a new object. example new JFrame("PNG Viewer").

primitive type: basic data like integer, boolean and holds the actual value.

reference type: objects like String, JFrame, InputStream. holds a link to the real thing in memory.

package: a folder of related classes.
